# Bettada Huvu Tours & Travels

A modern, animated tourism web application featuring a React.js frontend and a Java Spring Boot backend.

## Architecture
- **Frontend**: React (Vite), Tailwind CSS v4, Framer Motion
- **Backend**: Java 17, Spring Boot, Spring Data JPA, H2 Database

## Getting Started

### Frontend
1. Navigate to the `frontend/` directory.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server on `http://localhost:5173`.

### Backend
1. Navigate to the `backend/` directory.
2. Run `./gradlew bootRun` (or `.\gradlew.bat bootRun` on Windows) to start the API on `http://localhost:8080`.

## Features
- Fully responsive, animated UI matching the brand logo.
- 3D-enhanced layout and dynamic navigation.
- Real-time fetching of Tour Packages and Contact Inquiry REST API integration.
